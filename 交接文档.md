## 一、这是什么(背景)

- 原版:C#/WPF + .NET 8 的 Windows 本地待办(单文件 exe 63MB),**完整保留在 `legacy/`**,是迁移/对齐的参照物。
- 现版:**Tauri v2 + Rust(rusqlite/SQLite WAL)+ React 19 + TS + Zustand v5(无路由)+ Tailwind v4 + pragmatic-drag-and-drop + auto-animate**,便携 exe ≈10MB(需系统 WebView2)。
- 架构/命令/约定/坑见 **`CLAUDE.md`**,技术清单见 **`docs/tech-stack.md`**(改依赖/架构要同步更新它——见记忆 `keep-tech-stack-doc-updated`)。

## 六、开发与自测

- 跑:`npm run tauri dev`。**`cargo` 不在默认 PATH**,先 `export PATH="$USERPROFILE/.cargo/bin:$PATH"`(bash)。改 Rust 自动重编重启;改 `tauri.conf.json`/`capabilities` 要完全重启。残留:`taskkill /F /IM minimal-todo.exe`,端口 1420 被占就杀对应 PID。
- 数据库 `%AppData%\MinimalTodoApp\todo.db`(已导入用户真实数据)。预览某主题/视图最稳:停应用 → 用 python 写 `settings` 表(`theme` / `selected_group_id` 等)→ 重启。
- **截图自测的坑**(都踩过):
  - 合成鼠标点击/拖拽不可靠,关键交互(拖拽、文件对话框、动画手感)**让用户手测**。
  - **DPI**:PowerShell 截图默认逻辑坐标,`GetWindowRect` 给物理坐标 → 错位/假“透明边框”。截图脚本先 `SetProcessDPIAware()` 让两者都用物理坐标。
  - **贴边自动隐藏**会把窗口停到屏幕外(`Top` 很大的负数);测试前可删 `settings.dock_edge` 或把窗口 `SetWindowPos` 拉回可见处。
  - IME 常把英文合成成中文(自动输入便签正文会乱)。
