## MinimalTodoApp v1.1.3

### English

This re-release polishes the input placeholder and reworks the Settings window.

- **Centered input placeholder** — the "add a task" hint text used to sit slightly above center in the input bar; it is now vertically centered.
- **Draggable Settings window** — the borderless Settings window can now be moved by dragging anywhere on it. Changing the font previews live (task/body text updates immediately).
- **Grouped Settings** — Settings is now a two-pane layout: a left nav switches between **General** and **Font** groups; the right side shows the matching panel.
- **Restore defaults** — the General group has a new "Restore defaults" button. On first run, or when you click it, the appearance resets to Microsoft YaHei, size 12, line spacing 1.3, checkbox 16. Existing users' saved fonts are untouched.
- **Narrower window** — the minimum window width is reduced (380 → 320), so you can shrink the app further.

Data stays fully compatible. Version remains 1.1.3.

Download `MinimalTodoApp-v1.1.3-win-x64.exe` and just double-click — no .NET runtime required (self-contained single file).

### 简体中文

本次重新发布优化了输入框提示文本，并重做了设置窗口。

- **提示文本居中** —— 「添加新任务」的占位提示原先在输入栏里偏上，现已垂直居中。
- **设置窗口可拖动** —— 无边框的设置窗口现在可以在任意位置按住拖动；修改字体支持实时预览（正文/任务文字即时更新）。
- **设置项分组** —— 设置窗重做为左右两栏：左侧导航在「常规」与「字体」两组间切换，右侧显示对应面板。
- **恢复默认设置** —— 常规组新增「恢复默认设置」按钮。首次运行该软件或点击该按钮时，外观会重置为微软雅黑、字号 12、行距 1.3、勾选框 16；老用户已保存的字体设置不受影响。
- **窗口可更窄** —— 窗口最小宽度由 380 缩小到 320，可以把应用拖得更窄。

数据完全兼容，版本号维持 1.1.3。

下载 `MinimalTodoApp-v1.1.3-win-x64.exe` 双击即可运行，无需安装 .NET 运行时（自包含单文件）。
