# v2.0.2

在 v2.0.1 基础上,便签编辑器大升级——对标 Obsidian / Notion 的常用能力,并打磨「显示为源码」与全局弹窗体验。

## 便签编辑器(大升级)

- **全局搜索**:第二侧栏搜索框,标题 + 正文搜索(可开正则),Ctrl+F 聚焦;结果带片段,点击直接打开。
- **插入超链接**:工具栏按钮 + Ctrl+K 地址面板,Markdown `[文字](url)` 正常往返。
- **撤销 / 重做**:工具栏按钮(不可用时自动置灰)。
- **引用 / 代码块**:工具栏一键插入。
- **文档大纲(TOC)**:右上角悬浮面板列出 H1–H3,点击滚动定位;可一键隐藏 / 显示并记住。
- **斜杠命令**:正文输入 `/` 弹快捷菜单(标题 / 列表 / 任务 / 引用 / 代码块 / 分割线 / 表格 / 图片)。
- **字数 + 最后修改时间**:编辑区底部状态栏实时显示。
- **回收站**:删除便签改为软删除,第二侧栏「回收站」可恢复 / 彻底删除 / 清空,不会自动清理。
- **导出 HTML**:便签右键「导出为 HTML」(内联样式、图片内嵌),浏览器打开后可打印另存为 PDF。

## 显示为源码(就地可编辑)

- 工具栏「显示为源码」**就地**把正文换成可编辑的 Markdown 源码(`#`、`>`、`**` 等符号可直接改),工具栏与平时完全一致,再点切回所见即所得。
- 切到源码时**光标精确保留**(不再跳行首),并把光标所在行**垂直居中**;源码态下右上角大纲依然保留。

## 体验改进

- **所有弹窗文案改自动换行**:确认框 / 更新提示 / 各下拉弹层 / Toast 通知等不再用省略号截断,长文本(长路径、报错原因等)完整换行显示。

# v2.0.1

基于 v2.0.0 的稳定性与体验改进。

## 剪切板(大幅增强)

- **支持图片**:系统截图 / 复制图片自动入库;从资源管理器**复制图片文件**也能识别并存为图片项(列表显示缩略图)。
- **类型筛选 + 日期筛选 + 搜索**:工具栏可按「文字 / 图片」筛选、按日期范围(今天 / 近 7 天 / 近 1 月 / 近 3 月 / 自定义)筛选,筛选面板默认展开可收起。
- **重复内容移到最前**(默认开,可关):复制 / 捕获已存在的内容时移到最新位置并删除历史重复,不再堆叠一模一样的条目。
- **默认分组过期清理**:可设 7 天 / 1 月 / 3 月 / 1 年 / 永不;只清理「默认分组」(未分组)里的过期项,已分组与置顶项永久保留。
- **图片预览**:单击缩略图即全屏看大图(点击 / Esc 关)。
- **浏览模式**(相册式):一次铺满展示一条,**滚轮 / ←→ 左右翻看**(最左为最新),图片 / 文本都铺满。
- **右键菜单**:预览 / 复制 / 编辑 / **另存为**(文本→`.txt`、图片→`.png`)/ 加入待办 / 加入便签 / 分组;列表与浏览模式通用。
- 设置里「剪切板」**独立成一个分组**(过期时间、去重开关)。

## 便签

- **拖入支持更多文件**:`.txt / .sql / .json` 等常见文本与代码文件(非 Markdown 自动按代码块保形导入)。
- **表格**:工具栏「插入表格」按钮(输入行 / 列生成,可拖列宽);**直接粘贴 GFM 管道表格**(`| --- | :--: |`)也会自动渲染成表格(支持列对齐 / emoji)。
- **单篇导出为 Markdown**(右键便签 → 导出)。
- **选中文本右键加入待办**(多行按行拆成多条)。
- **定位到当前便签**:第二侧栏头部按钮,一键展开当前便签所在分组并滚动到它。
- 修复:从便签复制文本时**每行末尾多出空格 / 多出空行**。
- 更新 / 重装说明里的 Markdown(`#`、`-`、`**粗体**`)现在会正常渲染。

## 自动更新 / 重新安装(重点修复)

- 资产下载改在本地后端完成,绕开 GitHub 资产 CDN 的跨域限制(此前点更新无进度、直接失败)。
- 安装照搬旧版 WPF 逻辑:**下载到独立文件、绝不覆盖正在运行的 exe**;新版启动即接管旧版。
- **打开新下载的 exe 会自动接管正在运行的旧实例**(用 Windows 重启管理器找出占用数据库的旧进程并关闭),旧文件随即可删、看到的就是新版。
- 旧进程退出加硬退兜底:更新后不再残留旧窗口 / 旧设置弹窗。
- 重装对话框打磨:标题正确、去掉多余的「正在检查更新」提示与底部关闭按钮。

## 新增

- **全局快捷键**:Alt+1 便签 / Alt+2 剪切板 / Alt+3 标签看板 / Alt+4 四象限 / Alt+5 全部待办 —— 召唤窗口并切到对应视图;可在「设置 → 通用」自行修改。召唤=从隐藏/最小化恢复并浮到最前(不居中、不常驻置顶);贴边收起态也能唤出。
- **手动下载**入口(更新/重装对话框内=当前版本;设置「关于」内=最新版本),应用内更新失败时的兜底。
- **检查更新**移到设置内,带「检查中 / 已是最新 / 检查失败」明确反馈 + 启动检查节流。
- **标签看板**每个标签分组底部可直接输入、回车新建待办(对齐旧版)。

## 改进 / 修复

- 主窗口**永不在任务栏显示**,只在通知栏托盘(双击托盘唤出)。
- 贴边自动隐藏修复:双屏不再贴到屏间共享边(避免失灵)、多次隐藏后不再失效、贴边收起也能用快捷键唤出。
- 剪切板:「默认」分组右键清空、新建分组同名自动加序号(新分组2 / 新分组3…)、第二侧栏可拖到更窄。
- 修复贴边自动隐藏在「窗口比显示器还宽」(窄副屏 / 高 DPI)时崩溃、导致自动隐藏失灵。
- 待办 / 剪切项鼠标悬停时轻微抬起 + 投影高亮(各界面版式统一)。
- 应用内更新 / 重装下载更稳:自动走系统代理(对齐浏览器)、加连接 / 读超时、失败自动重试,修复「下载卡在 0%」假死(超时后报错可退回手动下载)。

# v2.0.0

整体重写:C#/WPF → Tauri v2(Rust + rusqlite/SQLite WAL + React 19 + TypeScript)。
包体 63 MB → 约 10 MB,毫秒级启动,数据即改即存。首次启动自动导入旧版 `data.json`(任务/标签/便签/自定义主题/全部设置),原文件保留;旧版可随时回退。

## 任务

- 子待办、优先级、截止倒计时、周期提醒、置顶、六种排序、拖拽重排
- **拖拽支持改父级**:拖到两条之间时层级自动对齐下面那条,连带整棵子树移动
- 父子完成逻辑对齐旧版:单独完成的子任务原地划线保留,整族完成才归入已完成
- 子任务进度可选「数字 / 直线 / 圆环(填充父勾选框)」
- 新建待办后可弹快捷设置小窗,直接改优先级 / 截止 / 周期提醒(设置可开)

## 外观

- **15 套主题**(浅色 / 深色 / 玻璃),中英双语,默认 Classic
- **9 套界面版式**(经典 / 极客 / 可爱 / 文档 / 流体 / 粗野 / 纯净白玻 / 微色调 / 双层面板)
- **优先级展示**独立设置(复选框环 / 左侧竖线 / 信号强度 / 圆点 / 无)
- **勾选框自定义**(形状 / 大小 / 粗细,改了即派生自定义版式)
- 窄窗父子排版:子任务轻微缩小 + 字重/颜色降级 + 阶梯对齐缩进;长标题自动换行、复选框对齐首行
- 字体 / 字号 / 行距可调

## 视图

- 列表 / 已完成 / 标签看板 / 四象限 / **剪贴板** / 便签(Markdown)
- 标签二级侧边栏,支持拖待办到分组归类
- 日历:右侧并排弹出,日/周/月,国内法定节假日,拖待办设截止

## 剪贴板(类 Win+V / Ditto)

- 后台监听系统剪贴板,自动记录历史;第二侧栏「分组」(默认 + 自定义,显示条数)
- 剪贴项右键:加入待办 / 加入便签 / 复制 / 编辑(独立窗口编辑器,手动保存)
- 拖剪贴项到分组归类(单分组)、搜索剪贴历史
- 注:本版**图片捕获暂停,仅记录文本**

## 便签

- Markdown 编辑;**拖入 `.md` 文件即导入**(任意界面拖入归「导入」分组)
- 第二侧栏分组,折叠态镜像树

## 提醒与提示音

- 周期提醒;**最小化时也在右下角弹系统通知**
