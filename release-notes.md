## MinimalTodoApp v1.1.5

### English

A major theme overhaul: a dedicated theme window, grouped palettes (12 per style), a visual color picker, and a batch of readability / rounded-corner fixes.

- **New theme window** — "Theme" now opens a separate, **draggable and resizable** window that sits beside the app (no longer covers it). Click any swatch for **instant live preview**; it stays open so you can keep trying.
- **Grouped palettes, ~90 themes** — Favorites / Classic / Morandi / Macaron / Dunhuang / Mondrian / Memphis / Rococo / Matisse / Transparent. Each named group has **12 colorways**. **Favorites** shows your current theme plus the most-recently-used, sorted by last use.
- **Quick group bar** — click a group chip to jump straight to it (no dropdown).
- **Custom themes: create / edit / delete** — right-click a custom swatch to edit or delete it, and pick colors with a built-in **HSV color picker** (no more typing hex codes).
- **Readability** — every palette is contrast-checked, including right-click menus and the calendar. The calendar now **re-themes instantly** when you switch themes (fixes dim text under dark themes).
- **Rounded corners** — fixed the square corners on right-click menus; the date picker / calendar are themed and rounded too.

Download `MinimalTodoApp-v1.1.5-win-x64.exe` and just double-click — no .NET runtime required (self-contained single file).

### 简体中文

本次为主题系统的一次大改：独立主题窗口、按风格分组（每组 12 套）、可视化取色器，以及一批可读性 / 圆角修复。

- **独立主题窗口** —— “修改主题”现在弹出一个**可拖动、可调整大小**的独立窗口，停靠在主程序旁边（不再遮挡）。单击任一色板即**实时预览**，窗口保持打开可连续切换。
- **分组配色，约 90 套主题** —— 常用 / 经典 / 莫兰迪 / 马卡龙 / 敦煌 / 蒙德里安 / 孟菲斯 / 洛可可 / 马蒂斯 / 透明，命名分组**各 12 套**。“常用”先显示当前主题，再按上次使用排序列出最常用的主题。
- **分组快速选择栏** —— 点击分组标签直接跳到该分组（不是下拉框）。
- **自定义主题：新建 / 编辑 / 删除** —— 在自定义色板上右键可编辑或删除；选色用内置 **HSV 取色器**（不再手输十六进制）。
- **可读性** —— 每套配色都经对比度校验，包含右键菜单与日历。切换主题时日历会**实时重新着色**（修复深色主题下日历文字发暗看不清）。
- **圆角** —— 修复右键菜单残留的直角；日期选择器 / 日历也已主题化并圆角。

下载 `MinimalTodoApp-v1.1.5-win-x64.exe` 双击即可运行，无需安装 .NET 运行时（自包含单文件）。
