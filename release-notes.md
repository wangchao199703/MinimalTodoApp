## MinimalTodoApp v1.1.3

### English

Group icons get polish, the Glass theme is fixed up, and the parent/child task layout is redesigned to be more compact.

- **Group icons** — the standard groups (All / Completed / Work / Study / Life) now start with sensible default icons; when creating a new group you can pick its icon right in the input row; two more icon categories were added (Travel, Symbols) plus custom image import.
- **Glass theme fixes** — the window now has rounded corners (Win11), and right-click menus / Settings / dialogs use an opaque background so they're readable through the blur.
- **Subtasks sit under their parent** — adding a subtask now places it directly beneath its parent, in any sort order.
- **Compact parent/child layout** — a parent is a single line with its "Subtasks n/m" badge moved to the right end; subtasks are indented narrower cards with smaller text and tighter spacing, while keeping the checkbox the same distance from the card's left edge as the parent.
- **Help** — the "Contact" section now shows the project's GitHub address.

Plus everything else in 1.1.3 (card layout, circular priority checkbox, adjustable fonts, etc.). Data stays fully compatible.

Download `MinimalTodoApp-v1.1.3-win-x64.exe` and just double-click — no .NET runtime required (self-contained single file).

### 简体中文

分组图标进一步完善、毛玻璃主题修复、父子任务布局重新设计得更紧凑。

- **分组图标** —— 标准分组（所有待办 / 已完成 / 工作 / 学习 / 生活）现在带合适的默认图标；新建分组时可在输入行直接选图标；新增「出行」「符号」两个图标分类，并支持导入自定义图片。
- **毛玻璃主题修复** —— 窗口四角圆角（Win11），右键菜单 / 设置 / 各对话框改用不透明背景，透过模糊也清晰可读。
- **子任务紧贴父任务** —— 新增子任务会直接出现在父任务下方，任何排序都生效。
- **更紧凑的父子布局** —— 父任务单行显示、「子任务 n/m」移到行末；子任务是缩进、更窄、字号更小、间距更紧的卡片，同时保持「勾选框到卡片左边框的距离」与父任务一致。
- **使用说明** —— 「联系开发者」改为显示本项目的 GitHub 地址。

以及 1.1.3 的其它改进（卡片布局、圆形优先级勾选框、可调字体等）。数据完全兼容。

下载 `MinimalTodoApp-v1.1.3-win-x64.exe` 双击即可运行，无需安装 .NET 运行时（自包含单文件）。
