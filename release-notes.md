## MinimalTodoApp v1.1.3

### English

This re-release adds a proper application icon — a blue-to-indigo gradient rounded square with a white checkmark.

- **App icon** — the app now has a real checkmark icon shown on the EXE file, the taskbar, Alt-Tab, the window title bar, and the system-tray icon (all from the same `app.ico`). Previously the EXE/taskbar/window used the blank default WPF icon.

Everything else from 1.1.3 is unchanged (group icons, Glass theme fixes, compact parent/child layout, adjustable fonts, etc.). Data stays fully compatible.

Download `MinimalTodoApp-v1.1.3-win-x64.exe` and just double-click — no .NET runtime required (self-contained single file).

### 简体中文

本次重新发布为应用补上了正式图标 —— 蓝紫渐变圆角方块 + 白色对钩。

- **应用图标** —— 现在 EXE 文件、任务栏、Alt-Tab、窗口标题栏以及系统托盘都显示同一枚对钩图标（均来自 `app.ico`）。此前 EXE/任务栏/窗口用的是 WPF 默认空白图标。

1.1.3 的其它内容保持不变（分组图标、毛玻璃主题修复、紧凑父子布局、可调字体等）。数据完全兼容。

下载 `MinimalTodoApp-v1.1.3-win-x64.exe` 双击即可运行，无需安装 .NET 运行时（自包含单文件）。
