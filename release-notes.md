# v2.0.0

整体重写:C#/WPF → Tauri v2(Rust + rusqlite/SQLite WAL + React 19 + TypeScript)。
包体 63 MB → 约 10 MB,毫秒级启动,数据即改即存。首次启动自动导入旧版 `data.json`(任务/标签/便签/自定义主题/全部设置),原文件保留;旧版可随时回退。

## 任务

- 子待办、优先级、截止倒计时、周期提醒、置顶、六种排序、拖拽重排
- **拖拽支持改父级**:拖到两条之间时层级自动对齐下面那条,连带整棵子树移动
- 父子完成逻辑对齐旧版:单独完成的子任务原地划线保留,整族完成才归入已完成
- 子任务进度可选「数字 / 直线 / 圆环(填充父勾选框)」
- 新建待办后可弹快捷设置小窗,直接改优先级 / 截止 / 周期提醒(设置可开)

## 外观

- **15 套主题**(浅色 / 深色 / 玻璃),中英双语,默认 Classic
- **9 套界面版式**(经典 / 极客 / 可爱 / 文档 / 流体 / 粗野 / 纯净白玻 / 微色调 / 双层面板)
- **优先级展示**独立设置(复选框环 / 左侧竖线 / 信号强度 / 圆点 / 无)
- **勾选框自定义**(形状 / 大小 / 粗细,改了即派生自定义版式)
- 窄窗父子排版:子任务轻微缩小 + 字重/颜色降级 + 阶梯对齐缩进;长标题自动换行、复选框对齐首行
- 字体 / 字号 / 行距可调

## 视图

- 列表 / 已完成 / 标签看板 / 四象限 / **剪贴板** / 便签(Markdown)
- 标签二级侧边栏,支持拖待办到分组归类
- 日历:右侧并排弹出,日/周/月,国内法定节假日,拖待办设截止

## 剪贴板(类 Win+V / Ditto)

- 后台监听系统剪贴板,自动记录历史;第二侧栏「分组」(默认 + 自定义,显示条数)
- 剪贴项右键:加入待办 / 加入便签 / 复制 / 编辑(独立窗口编辑器,手动保存)
- 拖剪贴项到分组归类(单分组)、搜索剪贴历史
- 注:本版**图片捕获暂停,仅记录文本**

## 便签

- Markdown 编辑;**拖入 `.md` 文件即导入**(任意界面拖入归「导入」分组)
- 第二侧栏分组,折叠态镜像树

## 提醒与提示音

- 周期提醒;**最小化时也在右下角弹系统通知**
- 完成音 / 周期提醒音可分别选风格(极简 / 游戏化 / 禅意 / 可爱),可试听

## 其它

- **数据存储位置可配置**:可把待办/便签/图标/剪贴板数据整体迁移到新目录(复制→校验→删,安全)
- 标签 / 便签 / 剪贴板分组右键「清空」
- 设置为独立窗口,与主窗口实时同步;**重新安装/修复当前版本**
- 自绘标题栏、系统托盘、贴边自动隐藏、置顶、开机自启、边缘缩放
- 托盘「显示并居中」一键复位窗口
- 基于 GitHub Release 的应用内自动更新(便携 exe 换壳重启)
